# 🚗 Apex Auto

> **Caucasus International University | Front-End Development 2026**
> A collaborative final project built from the ground up in just two weeks.

Apex Auto is a responsive, reactive React application designed to simulate a modern auto parts and vehicle browsing experience. This project served as our capstone for the Front-End Development course, challenging us to implement complex state management, dynamic UI rendering, and mock data integration within a tight two-week sprint.

### ✨ Key Features

- **Dynamic Product Sliders:** Smooth, interactive components for browsing vehicles and products.
- **Complex Filtering Logic:** A robust car selection and filtering system for users to easily narrow down their search.
- **Shopping Cart State:** Fully functional cart logic to add, remove, and manage selected items using React state.
- **Responsive UI:** Tailored CSS ensuring optimal viewing across desktop, tablet, and mobile devices.
- **Mock API Integration:** Simulates real-world data fetching for a seamless front-end experience.

---

### 🛠️ Technologies Used

- **React & JSX:** For building reusable, reactive UI components.
- **CSS3:** Custom styling and responsive layouts.
- **Vite:** For fast development and optimized building.

### 🚀 Getting Started

If you'd like to run this project locally, follow these steps:

**1. Clone the repository**
`git clone https://github.com/LukaTonia/apex-auto.git`

**2. Navigate to the project directory**
`cd apex-auto`

**3. Install dependencies**
`npm install`

**4. Start the development server**
`npm run dev`

Open your browser and visit: `http://localhost:5173/apex-auto/`
